How It Works

Powered by real Machine Learning and Natural Language Processing

🗣️

Natural Language Input

NLTK tokenizes and lemmatizes your message. Symptom entities are extracted from free-form text — no forms to fill.

🤖

ML Diagnosis Engine

A RandomForestClassifier trained on 2,000+ synthetic patient samples maps your symptoms to conditions with confidence scores.

📊

Confidence Scoring

Each diagnosis shows a probability score so you understand how strongly the model supports each condition.

📅

Instant Appointment Booking

Doctors are filtered by the recommended specialty. Pick a slot, fill a form, and get confirmed — stored in SQLite.

🚨

Emergency Detection

Symptoms with severity score ≥9 (e.g. chest pain + seizure) trigger immediate emergency guidance.

🔄

Multi-turn Dialogue

The conversation state machine asks follow-up questions, collects age and duration, then refines the ML analysis.

Built with: 🐍 Python 🌶️ Flask 🌲 scikit-learn 📝 NLTK 🗄️ SQLite
